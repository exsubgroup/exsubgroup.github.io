📥 Installation

Install in under 2 minutes

Works on Chrome, Brave, Edge, and any Chromium browser.

1
Download the extension file

Click the Download Extension button above. A .crx or .zip file will be saved to your device.

2
Open Extensions page in Chrome

Type chrome://extensions in your address bar and press Enter. Make sure Developer Mode is toggled ON (top-right corner).

3
Load the extension

If you have a .zip file: unzip it first, then click Load Unpacked and select the unzipped folder.

If you have a .crx file: drag and drop it directly onto the chrome://extensions page.

4
Pin it to your toolbar

Click the puzzle-piece icon 🧩 in the Chrome toolbar, find Exsub Group, and click the pin icon so it always shows.

5
Enter your User Code

Click the Exsub extension icon in your toolbar. Enter your User Code to connect your account. You're done!
